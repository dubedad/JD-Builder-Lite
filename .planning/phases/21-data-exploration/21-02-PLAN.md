Create `src/services/parquet_reader.py` with two functions and a module-level cache:

```python
"""Parquet reader service with CoverageStatus error handling.

Wraps pandas read_parquet with three-state error handling (LOAD_ERROR,
NOT_FOUND, FOUND) and warning logging on every failure path.
"""

import logging
from pathlib import Path
from typing import Optional

import pandas as pd

from src.models.parquet import CoverageStatus, ParquetResult

logger = logging.getLogger(__name__)

# Module-level cache: path string -> DataFrame. Loaded once per process.
_cache: dict[str, pd.DataFrame] = {}
```

**Function 1: `read_parquet_safe(path: Path) -> ParquetResult[pd.DataFrame]`**

- If `not path.exists()`: log `logger.warning("Parquet file not found: %s", path)`, return `ParquetResult(status=CoverageStatus.LOAD_ERROR, error=f"File not found: {path}")`
- Try `pd.read_parquet(path)` in try/except. On ANY exception (ArrowInvalid, etc.): log `logger.warning("Failed to read parquet file %s: %s", path, str(e))`, return `ParquetResult(status=CoverageStatus.LOAD_ERROR, error=str(e))`
- On success: strip column whitespace with `df.columns = df.columns.str.strip()`, cache in `_cache[str(path)]`, return `ParquetResult(status=CoverageStatus.FOUND, data=df)`
- Check cache first: if `str(path) in _cache`, return cached DataFrame as FOUND immediately

**Function 2: `lookup_profile(path: Path, code_col: str, profile_code: str, data_cols: Optional[list[str]] = None) -> ParquetResult[pd.DataFrame]`**

- Call `read_parquet_safe(path)`. If not FOUND, return that result (propagates LOAD_ERROR).
- Filter: `matches = df[df[code_col] == profile_code]`
- If `matches.empty`: log `logger.warning("Profile %s not found in %s (column: %s)", profile_code, path.name, code_col)`, return `ParquetResult(status=CoverageStatus.NOT_FOUND, error=f"Profile {profile_code} not found in {path.name}")`
- If data_cols provided: `matches = matches[data_cols]`
- Return `ParquetResult(status=CoverageStatus.FOUND, data=matches)`

**Important implementation details:**
- Use `logger = logging.getLogger(__name__)` -- matches existing pattern in enrichment_service.py and other services
- Every non-FOUND path MUST call `logger.warning()` -- this is the DATA-04 requirement
- Column whitespace stripping happens inside `read_parquet_safe`, not in `lookup_profile` -- strip once at read time
- The cache key is `str(path)` to handle both Path and string inputs consistently
- Do NOT add a cache invalidation mechanism -- gold files change infrequently and the app restarts between updates
