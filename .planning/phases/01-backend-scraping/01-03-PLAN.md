<task type="auto">
  <name>Task 1: API routes blueprint</name>
  <files>
    src/routes/api.py
  </files>
  <action>
Create the API routes that handle search and profile requests.

Create `src/routes/api.py`:

1. Create Blueprint: `api_bp = Blueprint('api', __name__, url_prefix='/api')`

2. Import services:
   - from src.services.scraper import scraper
   - from src.services.parser import parser
   - from src.services.mapper import mapper

3. Import models:
   - from src.models.responses import SearchResponse, ProfileResponse, ErrorResponse
   - from src.models.noc import SourceMetadata

4. `@api_bp.route('/search')` - GET:
   - Get query param 'q' from request.args
   - Validate: if not q or len(q) < 2, return ErrorResponse with 400
   - Try:
     - html = scraper.search(q)
     - results = parser.parse_search_results(html)
     - Create SearchResponse with query, results, count, metadata
     - Return jsonify(response.model_dump())
   - Except requests.RequestException as e:
     - Log error
     - Return ErrorResponse("Search failed", detail=str(e)), 502
   - Except Exception as e:
     - Log error
     - Return ErrorResponse("Internal error"), 500

5. `@api_bp.route('/profile')` - GET:
   - Get query param 'code' from request.args
   - Validate: if not code, return ErrorResponse with 400
   - Validate: code matches NOC pattern (5 digits, optional .2 digits)
   - Try:
     - html = scraper.fetch_profile(code)
     - noc_data = parser.parse_profile(html, code)
     - jd_data = mapper.to_jd_elements(noc_data)
     - Create ProfileResponse from jd_data
     - Return jsonify(response.model_dump())
   - Except with same error handling as search

6. `@api_bp.route('/health')` - GET:
   - Return {"status": "ok", "version": "1.0.0"}
   - Used to verify server is running

Use Flask's current_app.logger for logging errors.

Create the Flask application that ties everything together.

Create `src/app.py`:

1. Imports:
   - from flask import Flask
   - from flask_cors import CORS
   - from src.routes.api import api_bp

2. Create app factory function `create_app()`:
   - app = Flask(__name__, static_folder='../static')
   - CORS(app) to enable cross-origin requests from frontend
   - app.register_blueprint(api_bp)
   - Return app

3. Main block for direct execution:
   ```python
   if __name__ == '__main__':
       app = create_app()
       app.run(debug=True, port=5000)
   ```

4. Also create module-level app for WSGI servers:
   ```python
   app = create_app()
   ```

The app factory pattern allows testing with fresh app instances.
