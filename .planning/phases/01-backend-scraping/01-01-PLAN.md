<task type="auto">
  <name>Task 3: OASIS scraper service</name>
  <files>
    src/services/scraper.py
  </files>
  <action>
Create the HTTP client for fetching from OASIS. Follow research patterns exactly.

Create `src/services/scraper.py` with class `OASISScraper`:

1. `__init__(self, timeout: int = None)`:
   - Import timeout default from config.REQUEST_TIMEOUT
   - Create requests.Session()
   - Set User-Agent header from config.USER_AGENT
   - Store timeout

2. `search(self, query: str, version: str = None) -> str`:
   - Use version from config.OASIS_VERSION if not provided
   - Build URL: `{OASIS_BASE_URL}/OaSIS/OaSISSearchResult`
   - Params: searchType="Keyword", searchText=query, version=version
   - Call session.get() with timeout
   - Call response.raise_for_status()
   - Return response.text (raw HTML)

3. `fetch_profile(self, code: str, version: str = None) -> str`:
   - Build URL: `{OASIS_BASE_URL}/OaSIS/OaSISOccProfile`
   - Params: code=code, version=version
   - Same pattern as search()
   - Return response.text

4. Add module-level singleton: `scraper = OASISScraper()` for easy import

Handle SSL: First attempt without special handling. If SSL errors occur in testing, we'll add certifi.where() in a later fix.

Do NOT add retry logic yet - keep it simple for v1.
  </action>

  </verify>
