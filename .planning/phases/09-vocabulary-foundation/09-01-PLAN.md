  <files>

  <action>
Create VocabularyValidator class in src/vocabulary/validator.py:

1. __init__(vocab_index: VocabularyIndex) - stores reference to vocabulary index

2. Define STOP_WORDS set (filter from coverage calculation):
   {"the", "a", "an", "and", "or", "of", "to", "in", "for", "on", "with", "is", "are", "was", "were", "be", "been", "being", "by", "as", "at", "from", "that", "this", "it"}

3. validate_text(text: str) -> dict method:
   - Tokenize text using regex: re.findall(r'\b\w+\b', text)
   - Filter out stop words (case-insensitive)
   - For remaining words, check each against vocab_index.is_noc_term()
   - Return dict with:
     - coverage_percentage: float (NOC words / total non-stop words * 100, rounded to 2 decimals)
     - total_words: int (total non-stop words)
     - noc_words: int (count of words in NOC vocabulary)
     - non_noc_words: List[str] (specific words not in NOC vocabulary)
   - Handle empty text: return coverage_percentage=0.0, total_words=0

4. Update src/vocabulary/__init__.py to export VocabularyValidator

Use casefold() for all comparisons. Regex pattern r'\b\w+\b' for word tokenization.
  </action>
