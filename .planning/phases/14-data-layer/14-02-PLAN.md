2. **create_scraping_session() function:**
   From RESEARCH.md verified pattern:
   ```python
   from requests.adapters import HTTPAdapter
   from urllib3.util.retry import Retry

   def create_scraping_session() -> requests.Session:
       session = requests.Session()

       retry_strategy = Retry(
           total=3,
           backoff_factor=2,  # Wait 2, 4, 8 seconds
           status_forcelist=[429, 500, 502, 503, 504],
           allowed_methods=["GET", "HEAD"]
       )

       adapter = HTTPAdapter(max_retries=retry_strategy)
       session.mount("https://", adapter)
       session.mount("http://", adapter)

       session.headers.update({
           'User-Agent': 'JD-Builder/1.0 (TBS Occupational Data Scraper)'
       })

       return session
   ```

3. **TBSHttpClient class:**
   - __init__(self, session: requests.Session = None):
     - Creates session if not provided
     - Tracks last_request_time for rate limiting

   - _rate_limit(self):
     - Calculate time since last request
     - Sleep if < RATE_LIMIT_DELAY
     - Update last_request_time

   - fetch(self, url: str) -> tuple[bytes, dict]:
     - Calls _rate_limit()
     - Makes GET request with timeout
     - Returns (response.content, response_metadata)
     - response_metadata = {
         'url': response.url,
         'status': response.status_code,
         'headers': dict(response.headers),
         'elapsed_ms': response.elapsed.total_seconds() * 1000
       }
     - Raises HTTPError on failure (after retries exhausted)

   - fetch_occupational_groups_table(self) -> tuple[bytes, dict]:
     - Convenience method: fetch(TBS_BASE_URL + OCCUPATIONAL_GROUPS_PATH)

   - fetch_definitions_page(self) -> tuple[bytes, dict]:
     - Convenience method: fetch(TBS_BASE_URL + DEFINITIONS_PATH)

   - fetch_definition_anchor(self, anchor: str) -> tuple[bytes, dict]:
     - Fetch definitions page (for specific group anchor like #def-ai)
     - Note: Full page fetched, anchor is for parsing context

Follow existing project pattern from src/services/scraper.py for session management.
Do NOT disable SSL verification (unlike OASIS scraper which uses verify=False).

Create `src/scrapers/html_archiver.py` for provenance tracking:

1. **Constants:**
   - ARCHIVE_DIR = Path("data/html_archive")

2. **calculate_content_hash(content: bytes) -> str:**
   - Uses hashlib.sha256
   - Returns hex digest string

3. **archive_html(content: bytes, url: str, timestamp: str) -> dict:**
   - Creates ARCHIVE_DIR if not exists
   - Calculates content hash
   - Generates filename: {url_slug}-{timestamp}-{hash[:8]}.html
     - url_slug = URL path sanitized (replace / with -, remove .html)
     - timestamp = ISO format with - instead of : for Windows compatibility
   - Writes content to file (binary mode, preserve encoding)
   - Returns provenance metadata dict:
     ```python
     {
         'url': url,
         'scraped_at': timestamp,
         'content_hash': content_hash,
         'archive_path': str(archive_path),
         'file_size_bytes': len(content)
     }
     ```

4. **content_changed(url: str, new_content: bytes, repository) -> bool:**
   - Calculate hash of new_content
   - Get last hash from repository.get_last_content_hash(url)
   - Return True if different or if never scraped before
   - Log change detection result

5. **get_archived_content(archive_path: str) -> bytes:**
   - Read and return archived HTML
   - For re-parsing after code changes

Follow RESEARCH.md Pattern 4 (Content-Addressed HTML Archiving).

Create `src/scrapers/tbs_parser.py` with BeautifulSoup parsers:

1. **parse_occupational_groups_table(html: bytes) -> List[dict]:**
   Parse the concordance table from occupational-groups.html.

   Based on WebFetch analysis, extract for each row:
   - group_code: Text from first column (abbreviation like "AI", "CP")
   - subgroup: Text from Subgroup column (may be empty)
   - definition_url: href from Definition column link
   - qualification_standard_url: href from Qualification standard column
   - job_evaluation_standard_url: href from Job evaluation standard column (for concordance)
   - rates_of_pay_represented_url: href from represented employees column
   - rates_of_pay_unrepresented_url: href from unrepresented column

   Use BeautifulSoup with lxml parser:
   ```python
   soup = BeautifulSoup(html, 'lxml')
   table = soup.find('table')  # Main concordance table
   rows = table.find_all('tr')[1:]  # Skip header
   ```

   Return list of dicts with extracted data.
   Log warning (don't fail) if row structure unexpected.

2. **parse_definition_page(html: bytes, group_code: str = None) -> List[dict]:**
   Parse definitions.html to extract all group definitions.

   Based on WebFetch analysis:
   - Groups are in sections with anchors like #def-ai
   - Definition text follows group heading
   - Inclusions in ordered list after "Inclusions" heading
   - Exclusions in ordered list after "Exclusions" heading

   For each group found, return:
   ```python
   {
       'group_code': group_code,  # From anchor or heading
       'subgroup': subgroup,       # May be None
       'definition': definition_text,
       'inclusions': [
           {'statement': text, 'order': 1},
           {'statement': text, 'order': 2},
           ...
       ],
       'exclusions': [
           {'statement': text, 'order': 1},
           ...
       ]
   }
   ```

   If group_code provided, return only that group's data.
   If None, return all groups found on page.

   Handle edge cases:
   - Missing Inclusions section: return empty list
   - Missing Exclusions section: return empty list
   - Subgroup definitions: identify parent group
   - Log warnings for parse failures, don't crash

3. **parse_allocation_guide(html: bytes) -> List[dict]:**
   Parse the allocation guide and label paragraphs for provenance.

   Return list of:
   ```python
   {
       'paragraph_label': 'P1',  # Sequential label
       'content': paragraph_text,
       'section': section_heading  # Parent heading
   }
   ```

   Used for CONTEXT.md requirement: "Label and store TBS allocation guide paragraphs as P1, P2, P3..."

4. **Validation helper - validate_parsed_group(group: dict) -> List[str]:**
   - Check group_code is non-empty
   - Check definition is non-empty
   - Return list of validation errors (empty if valid)

Use error handling per RESEARCH.md:
- Wrap BeautifulSoup selectors in try/except
- Return None or empty list on failure, log error
- Never crash entire scrape for one bad section
