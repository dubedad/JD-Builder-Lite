  <name>Task 2: Create generation service with retry logic and fallback</name>
  <files>src/services/generation_service.py</files>
  <action>
Create `src/services/generation_service.py` implementing the full generation lifecycle per RESEARCH.md patterns:

1. **Imports:**
   - openai.OpenAI client
   - tenacity: retry, stop_after_attempt, wait_random_exponential, retry_if_exception_type
   - VocabularyValidator from src.vocabulary.validator
   - VocabularyIndex from src.vocabulary.index
   - get_high_quality_examples from src.services.few_shot_examples
   - STYLE_RULES from src.services.style_constants
   - check_semantic_equivalence from src.services.semantic_checker
   - Models: StyledStatement, GenerationAttempt, StyleVersionHistory from src.models.styled_content
   - Models: StyleContentType from src.models.ai
   - Models: VocabularyAudit, VocabularyTerm from src.models.vocabulary_audit
   - datetime, uuid, math, logging

2. **Custom Exceptions:**
   - VocabularyValidationError(Exception): raised when coverage < threshold
   - SemanticEquivalenceError(Exception): raised when similarity < threshold

3. **Configuration Constants:**
   - MAX_RETRIES = 3 (per RESEARCH.md recommendation)
   - VOCABULARY_COVERAGE_THRESHOLD = 95.0 (95% NOC words required)
   - STYLE_PROMPT_VERSION = "v1.0" (for provenance tracking, per existing pattern in llm_service.py)

4. **GenerationService class:**

   __init__(self, vocab_index: VocabularyIndex):
   - Store vocab_index
   - Create VocabularyValidator(vocab_index)
   - Initialize OpenAI client from OPENAI_API_KEY
   - Set up logger

   build_few_shot_prompt(self, section: str, noc_statement: str, n_examples: int = 5) -> tuple[str, str]:
   - Get style rule from STYLE_RULES.get(section, {})
   - Get high-quality examples: get_high_quality_examples(section, min_weight=0.85)[:n_examples]
   - Build system prompt per RESEARCH.md Pattern 1:
     * Role: "Canadian federal job description stylist"
     * Guidelines: follow section pattern, use ONLY NOC vocabulary, preserve meaning, professional style
     * Output: single complete sentence
   - Build user prompt with few-shot examples in format:
     "NOC Input: {ex['noc_input']}\nStyled Output: {ex['styled_output']}"
   - Return (system_prompt, user_prompt)

   _create_vocabulary_audit(self, validation_result: dict) -> VocabularyAudit:
   - Convert validation_result from VocabularyValidator.validate_text() to VocabularyAudit model
   - Create VocabularyTerm objects for NOC terms (is_noc_term=True)
   - Note: VocabularyValidator doesn't return term categories, so use "unknown" for category field

   _calculate_confidence(self, logprobs_data) -> float:
   - Per RESEARCH.md Pattern 4: mean token probability from logprobs
   - Handle None logprobs (return 0.5 default)
   - Convert log probs to probs: math.exp(logprob)
   - Return mean, clipped to 0-1

   @retry decorator with:
   - wait=wait_random_exponential(min=1, max=10)
   - stop=stop_after_attempt(MAX_RETRIES)
   - retry=retry_if_exception_type((VocabularyValidationError, openai.RateLimitError, openai.APITimeoutError))

   _generate_with_validation(self, section: str, noc_statement: str) -> dict:
   - Build prompts via build_few_shot_prompt()
   - Call OpenAI chat.completions.create with:
     * model="gpt-4o" (same as llm_service.py)
     * temperature=0.3 (lower than overview's 0.7)
     * max_tokens=200
     * logprobs=True, top_logprobs=1
   - Extract styled_text from response
   - Validate vocabulary: self.validator.validate_text(styled_text)
   - If coverage < VOCABULARY_COVERAGE_THRESHOLD, raise VocabularyValidationError
   - Return dict with styled_text, validation_result, logprobs

   generate_styled_statement(self, noc_statement_id: str, noc_text: str, section: str) -> StyledStatement:
   - Per RESEARCH.md Pattern 5: orchestrate with fallback
   - Try:
     * Call _generate_with_validation()
     * Check semantic equivalence via check_semantic_equivalence()
     * If not equivalent, raise SemanticEquivalenceError
     * Calculate confidence from logprobs
     * Create VocabularyAudit from validation result
     * Return StyledStatement with content_type=StyleContentType.AI_STYLED
   - Except (VocabularyValidationError, SemanticEquivalenceError, tenacity.RetryError):
     * Log the failure
     * Create fallback VocabularyAudit (100% coverage, empty non_noc_terms)
     * Return StyledStatement with:
       - styled_text = noc_text (original)
       - content_type = StyleContentType.ORIGINAL_NOC
       - confidence_score = 1.0
       - retry_count = MAX_RETRIES
       - vocabulary_coverage = 100.0

5. **Module-level singleton pattern:**
   - _generation_service: Optional[GenerationService] = None
   - get_generation_service(vocab_index: VocabularyIndex) -> GenerationService
   - This allows lazy initialization with the vocab_index from app startup

Important: Use existing OPENAI_API_KEY and OPENAI_MODEL patterns from src/config.py and src/services/llm_service.py. Temperature 0.3 is per RESEARCH.md recommendation.
