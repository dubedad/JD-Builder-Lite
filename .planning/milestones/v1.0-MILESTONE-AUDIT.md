
## Tech Debt Summary

**5 items across 3 areas — no blockers**

### Priority: Address Before Next Pipeline Re-Run

**DB_PATH divergence** — if `python pipeline/ingest.py` or `enrich.py` or `bridge.py` is re-run, the app will silently serve stale data. Fix: update `DB_PATH` in `main.py` to point to the root `careers.sqlite`:

```python
# main.py line 19 — change:
DB_PATH = os.path.join(os.path.dirname(__file__), "pipeline", "careers.sqlite")
# to:
DB_PATH = os.path.join(os.path.dirname(__file__), "careers.sqlite")
```

### Low Priority (Cosmetic)

- Footer quick-links (`/careers?function=digital` etc.) don't pre-filter — remove or implement query param support
- Blank Job Function dropdown option — add `WHERE job_function != ''` to the DISTINCT query in `/careers` route
- `related_careers` column queried but never rendered — remove from `SELECT` in L3 route, or implement display
