## Detailed Requirements

### A.1 Search Bar — MAJOR REDESIGN

**Scope:** Complete welcome page redesign

**Changes:**
1. Use OaSIS welcome page layout (image left, search right)
2. Title: "Welcome to JobForge Job Description Builder"
3. LLM-generated description below title
4. "Explore JobForge" section header
5. Replace 5 OaSIS modules with 5 new JobForge modules

**5 New Modules:**

| # | Module | Description | Icon (semantic) |
|---|--------|-------------|-----------------|
| 1 | WiQ Workforce Intelligence Platform | Enable business and data management teams to have a conversation with the workforce intelligence semantic model and knowledge graph. Features: load WiQ Golden Model, Export WiQ Data Catalogue, GC HR Data Dictionary, MS Purview Export, ask workforce planning questions, provide data management teams with WiQ architectural and data surfacing specifications. | TBD |
| 2 | Data Governance | Medallion certified provenance complying with Directive on Automated Decision Making, traceability of WiQ and WiQ HR artifacts to requirements, audit trail showing decisions tied to authoritative sources, data stewards, publication dates, raw data table provenance. | TBD |
| 3 | Job Notice and SOMC Creator | Automates creation of Job Notice in same format as PSRS (https://emploisfp-psjobs.cfp-psc.gc.ca/psrs-srfp/applicant/page2440) | TBD |
| 4 | Job Classification Helper | Takes job descriptions and classifies them using 'dry classification' which automates the process using classification directive and guidance. | TBD |
| 5 | GC Job Architecture | Explore the GC Job Architecture - like NOC but for GC Job Titles, not private sector labor market intelligence. | TBD |

**Module behavior:**
- Use "Learn more..." format from OaSIS
- Semantic matching icons
- Hover effects (light up like OaSIS)
- NO LINKS for now (placeholder modules)

**HTML Reference:** OaSIS Welcome page HTML provided
