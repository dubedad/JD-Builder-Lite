# v2.1 Release Notes

**Release Date:** 2026-02-02
**Tag:** v2.1

## Summary

v2.1 completes the UI redesign started in v2.0 and adds full TBS Directive compliance with OaSIS provenance tracking.

---

## Features

### UI Improvements
- **Profile Tabs**: NOC attributes organized by category (Job Elements, Reference Attributes, Additional Info)
- **Collapsible Accordions**: Sections within tabs for better organization
- **Card Layout**: Statements displayed as cards with descriptions and proficiency levels
- **Filter Chips**: Source attribute filtering for statement selection
- **Navy Theme**: Consistent color scheme across all pages

### Export Enhancements
- **Descriptions**: Element descriptions included in exported job descriptions
- **Proficiency Levels**: Importance ratings (L1-L5) displayed with visual indicators
- **Full Provenance**: Publication dates and source URLs for each statement
- **Compliance Table**: Job Factor, Statement, Source, Publication Date, URL columns
- **Audit Trail**: Selection timestamps for Directive 6.2.7 compliance

### Bug Fixes
- Fixed "Create Job Description" button not responding (event delegation issue)
- Fixed source_attribute display in exports
